B - Build Tree

Description

给定一个集合， 让你构建一棵 $m$ 层的满 $n$ 叉树，树的根为 $0$ 号点，同时边权从给定的集合里选择。
现在定义: $d(i)$ 为第 $i$ 号点到根的距离，现在要求如何分配边权，使 $\sum\limits_{i=0}^Nd(i)$ 最小，答案对 $p$ 取模

题解

$ans=\sum\limits_{i=1}^{m-1} 第i层边被计算次数*第i层边权总和$, $第i层边被计算次数=\sum\limits_{0}^{m-i-1} n^i $ , 并且将集合中的数从小到大分配给边权

C - Chika and Friendly Pairs

Description

给出一个序列 $(1\leq n \leq 27000)$ ,给出 $k$ , 并有 $m$ $(1 \leq m \leq 27000)$ 次询问 ，每次询问在 $[L,R]$ 中满足 $|a_i-a_j| <= k , i< j$ 的有序对个数

题解

发现增加或删除一个元素时的贡献比较好计算，那么使用莫队算法。
计算贡献需要查询 $[a_i-k,a_i+k]$ 的元素个数 ，可以使用动态开点权值线段树，
或者是树状数组+离散化 ($a_i,a_i-k,a_i+k$)

E - Hello XTCPC

Description

给你一个字符串 $(1\leq n \leq 2\times 10^5)$，询问最多能凑出多少个 “$xtCpc$”，每个位置最多使用一次

题解

贪心，每次选用尽量靠前的字符
