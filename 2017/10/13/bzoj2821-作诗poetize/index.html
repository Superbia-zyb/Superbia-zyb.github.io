[BZOJ2821] 作诗(Poetize)

Posted on 2017-10-13 Edited on 2020-07-05

题目描述

Description

神犇SJY虐完HEOI之后给傻×LYD出了一题：SHY是T国的公主，平时的一大爱好是作诗。由于时间紧迫，SHY作完诗 之后还要虐OI，于是SHY找来一篇长度为N的文章，阅读M次，每次只阅读其中连续的一段[l,r]，从这一段中选出一 些汉字构成诗。因为SHY喜欢对偶，所以SHY规定最后选出的每个汉字都必须在[l,r]里出现了正偶数次。而且SHY认 为选出的汉字的种类数（两个一样的汉字称为同一种）越多越好（为了拿到更多的素材！）。于是SHY请LYD安排选 法。LYD这种傻×当然不会了，于是向你请教……问题简述：N个数，M组询问，每次问[l,r]中有多少个数出现正偶 数次。

Input

输入第一行三个整数n、c以及m。表示文章字数、汉字的种类数、要选择M次。第二行有n个整数，每个数Ai在[1, c ]间，代表一个编码为Ai的汉字。接下来m行每行两个整数l和r，设上一个询问的答案为ans(第一个询问时ans=0)， 令L=(l+ans)mod n+1, R=(r+ans)mod n+1，若L>R，交换L和R，则本次询问为[L,R]。

Output

输出共m行，每行一个整数，第i个数表示SHY第i次能选出的汉字的最多种类数。

Sample Input

5 3 5
1 2 2 3 1
0 4
1 2
2 2
2 3
3 5

Sample Output

2
0
0
0
1

HINT

对于100%的数据，1<=n,c,m<=10^5

题目分析

分块 预处理出块和块之间出现正偶数次的数字数量 和块与块之间每种数字出现的数量

#include <cstdio>
#include <cstring>
#include <set>
#include <map>
#include <vector>
#include <cmath>
#include <queue>
#include <algorithm>
using namespace std;
int n,tot,m,siz,ans;
int val[100010],sum[320][100010],f[320][320],num[100010];
int sta[640];
void init()
{
    for(int i=0;i<n;i++) sum[i/siz][val[i]]++;
    for(int i=0;i*siz<n;i++)
    {
        for(int j=i*siz;j<n;j++)
        {   
            num[val[j]]++;
            if(num[val[j]]==2) f[i][j/siz]++;
            if(num[val[j]]==3) f[i][j/siz]--,num[val[j]]=1;
            if(j&&(j%siz==0)) f[i][j/siz]+=f[i][j/siz-1]; 
        }
        memset(num,0,sizeof num);
    }
    for(int i=1;i*siz<n;i++)
        for(int j=1;j<=tot;j++) sum[i][j]+=sum[i-1][j];

}
void getans(int x,int y)
{
    ans=0;
    if(x/siz==y/siz)
    {
        sta[0]=0;
        for(int i=x;i<=y;i++) sta[++sta[0]]=val[i];
        sort(sta+1,sta+sta[0]+1);
        int tmp=0;
        for(int i=1;i<=sta[0];i++)
        {
            if(sta[i]!=sta[i-1]) ans+=(tmp>0&&tmp%2==0)?1:0,tmp=0;
            tmp++;
        }
        ans+=(tmp>0&&tmp%2==0)?1:0;
        return ;
    }
    ans=f[x/siz+1][y/siz-1],sta[0]=0;
    for(int i=x;i<x/siz*siz+siz;i++) sta[++sta[0]]=val[i];
    for(int i=y/siz*siz;i<=y;i++) sta[++sta[0]]=val[i];
    sort(sta+1,sta+sta[0]+1);
    int tmp=0;
    for(int i=1;i<=sta[0];i++)
    {
        int nmp=sum[y/siz-1][sta[i-1]]-sum[x/siz][sta[i-1]];
        if(sta[i]!=sta[i-1])
        {
            if(nmp>0&&nmp%2==0) ans--;
            if((tmp+nmp)>0&&(tmp+nmp)%2==0) ans++;
            tmp=0;
        }
        tmp++;
    }
    int nmp=sum[y/siz-1][sta[sta[0]]]-sum[x/siz][sta[sta[0]]];
    if(nmp>0&&nmp%2==0) ans--;
    if((tmp+nmp)>0&&(tmp+nmp)%2==0) ans++;
}
int main()
{   
    scanf("%d%d%d",&n,&tot,&m),siz=sqrt(1.0*n);
    for(int i=0;i<n;i++) scanf("%d",&val[i]);
    init();
    for(int l,r,i=1;i<=m;i++)
    {
        scanf("%d%d",&l,&r);
        l=(l+ans)%n+1,r=(r+ans)%n+1;
        if(l>r) swap(l,r);
        getans(--l,--r),printf("%d\n",ans);
    }
    return 0;
}
