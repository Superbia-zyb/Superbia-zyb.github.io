[BZOJ2096] [Poi2010]Pilots

Posted on 2017-08-28 Edited on 2020-07-05

题目描述

Description

Tz又耍畸形了！！他要当飞行员，他拿到了一个飞行员测试难度序列，他设定了一个难度差的最大值，在序列中他想找到一个最长的子串，任意两个难度差不会超过他设定的最大值。耍畸形一个人是不行的，于是他找到了你。

Input

输入：第一行两个有空格隔开的整数k（0<=k<=2000,000,000），n(1<=n<=3000,000)，k代表Tz设定的最大值，n代表难度序列的长度。第二行为n个由空格隔开的整数ai（1<=ai<=2000,000,000），表示难度序列。

Output

输出：最大的字串长度。

Sample Input

3 9
5 1 3 5 8 6 6 9 10

Sample Output

4

（有两个子串的长度为4: 5, 8, 6, 6 和8, 6, 6, 9.最长子串的长度就是4）

题目分析

一开始写了个二分+RMQ 虽然时间复杂度对但是惨遭空间杀= = 没办法 只好单调队列+双指针

#include <cstdio>
#include <cstring>
#include <set>
#include <map>
#include <vector>
#include <cmath>
#include <queue>
#include <algorithm>
using namespace std;
int n,m;
int q[3000100];
int p[3000100];
int a[3000100];
int main()
{
    scanf("%d%d",&m,&n);
    for(int i=1;i<=n;i++) scanf("%d",&a[i]);
    int head=1,ans=0;
    int l1=1,l2=1,r1=0,r2=0;
    for(int i=1;i<=n;i++)
    {
        while(l1<=r1&&a[q[r1]]<=a[i]) r1--;
        q[++r1]=i;
        while(l2<=r2&&a[p[r2]]>=a[i]) r2--;
        p[++r2]=i;
        if(a[q[l1]]-a[p[l2]]<=m) ans=max(ans,i-head+1);
        else 
        {
            while(head<=i)
            {
                if(a[q[l1]]-a[p[l2]]<=m) break;
                head++;
                while(l1<=r1&&q[l1]<head) l1++;
                while(l2<=r2&&p[l2]<head) l2++;
            }   
        }
        if(head<=i) ans=max(ans,i-head+1);
    }
    printf("%d",ans);
    return 0;
}
